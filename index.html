Cómo usar el sistema

  1. Completa el formulario del costado y guarda un registro por turno o por día.
  2. En Historial revisa y elimina registros si es necesario.
  3. En Informe automático el sistema consolida comentarios del operador, acciones ejecutadas, pérdidas por agua, energía, mantención y conclusión operacional.
  4. Usa Exportar JSON para respaldo y traspaso entre computadores o celulares.
  5. Usa el archivo Excel del paquete para el control KPI de largo plazo.
  6. Los KPI muestran metros acumulados perforados calculados como perforaciones × largo unitario.
  7. Al importar respaldos antiguos, el sistema migra automáticamente los datos: si no existe largo unitario, toma el valor de avance cuando corresponde a un largo típico como 3.5 m.
Este sistema queda preparado para Boltec, Jumbo y Scoop. Puedes renombrar equipos, operadores y sectores sin tocar el código.

El informe resalta automáticamente cuando detecta agua en el sector o horas perdidas por agua.
